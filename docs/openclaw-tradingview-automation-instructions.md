## TradingView Automation Workflow

Repeat this workflow every 5 to 15 minutes during active monitoring.

1. Run the one-cycle agent command.
2. Confirm this file exists:

```text
C:\Users\ADMIN\Desktop\signal\outputs\tradingview\market_agent_zones.pine
```

3. Read the full contents of `market_agent_zones.pine`.
4. Open TradingView in a browser.
5. Open a forex chart, for example `NZDUSD`, `EURUSD`, `GBPJPY`, or `XAUUSD`.
6. Open TradingView's Pine Editor.
7. Replace the entire Pine Editor contents with the generated script.
8. Save or update the script if TradingView asks.
9. Click `Add to chart` or refresh the already-added indicator.
10. Verify the chart shows only user-facing labels such as:


Do not show or expose internal statuses such as:

```text

expired

```

If those internal labels appear on TradingView, stop and report that the Pine export filter needs fixing.

## Important Behavior

The same Pine Script can contain zones for multiple symbols.

TradingView will only display zones matching the currently open chart symbol.

Example:

- On `NZDUSD`, show only `NZDUSD` zones.
- On `GBPJPY`, show only `GBPJPY` zones.
- If a symbol has no valid current/future setup, show nothing for that symbol.

## Automation Loop

Recommended loop:

```text
Every 5 minutes:
1. Run agent once.
2. Read generated Pine Script.
3. If Pine Script changed since the previous cycle:
   - Update TradingView Pine Editor.
   - Add/refresh indicator.
4. If Pine Script did not change:
   - Leave TradingView alone.
5. Continue monitoring.
```

Use a file hash or exact text comparison to avoid unnecessary TradingView updates.

## If TradingView Blocks Automation

If TradingView shows login, captcha, ads, overlays, or a manual confirmation:

1. Stop automation.
2. Report the blocker.
3. Ask the user to handle the manual step.
4. Resume only after the chart and Pine Editor are accessible.

Do not repeatedly click random page elements.

## Verification Checklist

After each successful TradingView update, verify:

- The indicator is added to the chart.
- The labels are chart-facing and understandable.
- No internal AI status labels are visible.
- The currently open symbol matches the zones displayed.
- The latest Pine Script was generated after the latest agent run.
